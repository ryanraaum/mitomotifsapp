This site provides two web-based tools (seq2sites and sites2seq) to translate between human mitochondrial sequence data and variant sites relative to the revised Cambridge Reference Sequence (rCRS). Basic usage information is provided on the main page for each tool; further information is available on the documentation page.

seq2sites

seq2sites (sequence to sites) analyzes human mtDNA sequence data in order to determine the variant sites relative to the rCRS.

sites2seq

sites2seq (sites to sequence) transforms variant sites into sequence data.
