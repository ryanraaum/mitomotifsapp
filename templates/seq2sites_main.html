Input format

For 'Sequence Only', and 'Name and Sequence', enter sequences on separate lines. If using a name, that should be the first thing on each line, followed by a space (any number of spaces is ok).

If submitting sequences in FASTA format, simply paste in properly formatted FASTA sequences. Please note that there are limits on the number of sequences you may submit at one time. These limits are based on the size of the input range you choose below.

  • Single Sequence (See Example)
  • Multiple sequences in FASTA format (See Example)

Sequence input is in range:

Select the sequence range in which your sequences fall. It is important that your sequences fall completely within the range. For this reason, the options are fairly generous for their labels (i.e. the HVR1-named range here is substantially larger than the true HVR1 region). If in doubt, select 'All'.

It is also very important that your provided sequences are in the order of the input range. That is, if you have both HVR1 and HVR2 combined in the order HVR1, then HVR2, then you MUST select the 'HVR1 through HVR2' option. Alternatively (and unusually) if you have both HVR1 and HVR2, but have concatenated them in the order HVR2, then HVR1, then you must choose the 'All' range option. Finally, if you have sequences that run past the standard "end" of the mtDNA genome that are longer than the offered 'HVR1 through HVR2' range, you will have to cut and shuffle your sequences so that they fall in the 1-16569 range in that order.

  • HVR1 (positions 15993-16569)
  • HVR2 (positions 1-576)
  • HVR1 through HVR2 (positions 15993-576)
  • Coding Region (positions 577-15992)
  • All (positions 1-16569)
