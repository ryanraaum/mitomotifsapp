Input format

You may paste in either a single sequence (see example) or multiple sequences in FASTA format (see example). The first letter of your submission is used to determine what you have submitted. If the first letter is a '>' character, then it is assumed you have submitted multiple sequences in FASTA format. Otherwise it is assumed you have submitted a single sequence.

All non-IUPAC nucleotide characters will be removed before the sequence is processed, so it is ok if the sequences include spaces or line breaks. You should indicate unknown bases with 'N'.

If submitting multiple sequences, please note that there are limits on the number of sequences you may submit at one time.

Limits

The underlying implementation of this web service does not allow long-running requests, which means not too many sequences at once. The current limits are based upon the length of the sequences you submit and are:

  • Sequences <= {{ cutoff }}bp, up to {{ max_short }} sequences at a time.
  • Sequences > {{ cutoff }}bp, up to {{ max_long }} sequences at a time.
